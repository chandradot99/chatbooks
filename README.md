# Steps for Local Setup

1. **Clone Repository:**
   ```bash
   git clone git@github.com:chandradot99/chatbooks.git

2. **Install Ruby Gems:**
   ```bash
   bundle install

3. **Install Node Packages:**
   ```bash
   yarn

4. **Build UI Components:**
   ```bash
   bin/shakapacker-dev-server

5. **Run Rails Server:**
   ```bash
   bundle exec rails s -p 3000
